OwlAnim

Class responsible for animation handling

Constructor

new OwlAnim(options, type, animation)

Creates an animation

Parameters:
Name Type Attributes Description
options Object

options
type string | Element

If 'inSyntax' then the buttons will be displayed in the top-right corner of the atatched OwlSyntax element. If 'afterSyntax' then the buttons will be displayed after the atatched OwlSyntax element. If 'afterDraw' then the buttons will be displayed after the atatched OwlDraw element. Otherwise if string, the buttons will be the children of the DOM element with id matching this parameter. If instance of Element then the buttons will be children of this element.
options.draw string | OwlDraw <optional>

If string, then an empty OwlDraw instance will be created in the element with id matching this parameter, if OwlDraw instance then this instance will be used.
options.width number <optional>

Width of the svg canvas, if it needs to be created
options.height number <optional>

Height of the svg canvas, if it needs to be created
options.syntax string | OwlSyntax <optional>

If string, then an empty OwlSyntax instance will be created in the element with id matching this parameter, if OwlSyntax instance then this instance will be used.
options.language string | OwlSyntax.language <optional>

If string, then name of grammar (OwlSyntax[{name}]). If object, then it must be compatible with OwlSyntax.language. If undefined but the data-language property is set on the source element, then the language will be taken from that property.
options.buttons Object <optional>

Object describing the buttons. By default, start, next, continuous buttons are shown, continuousQuick is hidden.

The event handler that will be executed on button click. Note, that this function will be bounded to this OwlAnim instance (arrow functions can not be bounded, so use old style functions).
animation function

The generator function that implements the animation, It accepts an Object parameter. The object has two properties. The .draw which holds the OwlDraw instance if it is part of the animation, and .syntax which holds the OwlSyntax instance if it is part of the animation.
