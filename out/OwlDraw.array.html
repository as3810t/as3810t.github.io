Name Type Attributes Description
x number

x-coordinate of the first of circle (other circles will have 2*radius distance between each other)
y number

y-coordinate of the first of circle (other circles will have the same y-coordinate)
radius number

radius of circles
class string <optional>

list of class that will be added to the DOM element
stroke string <optional>

color of stroke, that applies to the circle
strokeWidth number <optional>

width of stroke, that applies to the circle
fill string <optional>

color of fill, that applies to the circle
type string

If 'param', the elements will be taken from the second parameter. If 'succeeding', the elements will be succeeding elements (e.g. 1, 2, 3, 4, ...). If 'random', the elements will be randomly generated. If 'empty', the elements will be all empty.
num number <optional>

Number of elements to create
min number <optional>

Minimum of numbers
max number <optional>

Maximum of numbers
sort string | function <optional>

If 'shuffle', the elements will be randomly shuffled (see shuffle). If 'sorted', the elements will be sorted ascendingly. If a function ((a,b)=>true|false), the elements will be sorted according to the function.
elements Array.<number> <optional>

Elements to be a base
