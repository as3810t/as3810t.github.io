Class that represents a binary tree

Constructor

Creates a binary tree with values

Parameters:
Name Type Attributes Description
options Object

Options

Properties
Name Type Attributes Description
tree string

If 'nice' then the tree will be consisted of OwlDraw.btElem, otherwise of OwlDraw.circle
width number

The width of the rectangle, the binary tree will be fit into
y number

the y-coordinate of the root element
radius number <optional>

The radius of elements if the tree is consisted of OwlDraw.circle.
size number <optional>

The width of the element if the tree is consisted of OwlDraw.btElem. The height of the data area matches this value, while the height of the pointer is the third of this value.
class string <optional>

list of class that will be added to the DOM element
type string

If 'param', the elements will be taken from the second parameter. If 'succeeding', the elements will be succeeding elements (e.g. 1, 2, 3, 4, ...). If 'random', the elements will be randomly generated. If 'empty', the elements will be all empty.
num number <optional>

Number of elements to create
min number <optional>

Minimum of numbers
max number <optional>

Maximum of numbers
sort string | function <optional>

If 'shuffle', the elements will be randomly shuffled (see shuffle). If 'sorted', the elements will be sorted ascendingly. If a function ((a,b)=>true|false), the elements will be sorted according to the function.
elements Array.<number> | Object <optional>

Elements to be a base

The right-side tree, with same structure as elements. If null, then no child.
